# smart-movie-recommendation-engine

## 📌 Project Overview
The **Movie Recommender System** is a machine learning-based web application that recommends movies similar to a selected movie.  
It uses a **similarity matrix** to find related movies and displays recommendations along with movie posters through an interactive **Streamlit** interface.

The system enhances user experience by providing personalized movie suggestions based on content similarity.

---

## 🚀 Features
- Recommend movies similar to the selected movie
- Interactive web interface using **Streamlit**
- Displays movie posters using **TMDB API**
- Fast recommendations using a **precomputed similarity matrix**
- User-friendly movie selection dropdown

---

## 🛠️ Tech Stack
- **Python**
- **Streamlit**
- **Pandas**
- **Pickle**
- **Requests API**
- **TMDB API**

---

## ⚙️ How It Works
1. Movie data is processed and converted into a feature representation.
2. A **similarity matrix** is generated to calculate similarity between movies.
3. When a user selects a movie, the system finds the most similar movies.
4. Movie posters are fetched using the **TMDB API**.
5. Results are displayed in the **Streamlit web interface**.
